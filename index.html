HTML Part

Web কি?

Web হচ্ছে তথ্যসমূহের ভান্ডার যেখানে প্রায় সকল ধরনের তথ্য থাকে।

HTML কি?

Hyper Text Markup Language.এটি কোন প্রোগামিং ল্যাংগুয়েজ নয়। এটি শুধুমাত্র একটি মার্কআপ ল্যাংগুয়েজ।

HTML 5 কি?

HTML 5 হচ্ছে HTML এর সর্বশেষ ভার্সন।

HTML 5 এর বেসিক স্ট্রাকচার?

CSS Part

CSS কি?

Cascading Style Sheets. এটি দ্বারা HTML এর Element কে স্টাইল করা হয়। যেমন, কালার, বর্ডার, মার্জিন, ব্যাকগ্রাউন্ট কালার, প্যাডিং, সাইজ, এলাইনম্যান্ট, উইড, হাইট ইত্যাদি নির্ধারন করা যায়।

CSS 3 কি?

CSS 3 হচ্ছে CSS এর সর্বশেষ ভার্সন।
